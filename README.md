# Building the Project

## Requirements

Apart from the Libraries managed by ivy, you need to have the following jars on your build path. 
They can be found in `$ANDROID_SDK/tools/lib` (tools rev. 21.0.1).

* chimpchat.jar
* common.jar
* sdklib.jar
